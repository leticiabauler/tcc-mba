# Formatação Básica

Nesta seção, você aprenderá os elementos fundamentais de formatação de texto em Markdown.

## Títulos

Markdown suporta 6 níveis de títulos:

# Título Nível 1
## Título Nível 2
### Título Nível 3
#### Título Nível 4
##### Título Nível 5
###### Título Nível 6

---

## Ênfase no Texto

### Itálico

Use *asteriscos* ou _underscores_ para itálico.

### Negrito

Use **dois asteriscos** ou __dois underscores__ para negrito.

### Negrito + Itálico

Combine ambos: ***três asteriscos*** ou ___três underscores___.

### Tachado

Use ~~dois tils~~ para texto tachado.

---

## Citações

Use `>` para criar citações em bloco:

> "A única maneira de fazer um ótimo trabalho é amar o que você faz."
> 
> — Steve Jobs

### Citações Aninhadas

> Citação de primeiro nível
>> Citação aninhada
>>> Citação ainda mais aninhada

---

## Links

### Link Simples

[Visite o GitHub](https://github.com)

### Link com Título

[PDI Learning Hub](https://pdi.example.com "Sistema de Aprendizado")

### Link de Referência

Este é um [link de referência][1] e este é [outro link][2].

[1]: https://github.com
[2]: https://example.com

---

## Linha Horizontal

Use três ou mais hífens, asteriscos ou underscores:

---

***

___

---

## Caracteres Especiais

Para exibir caracteres especiais literalmente, use backslash `\`:

\*Isto não é itálico\*

\**Isto não é negrito\**

---

## Código Inline

Use `crases` para código inline: `const x = 10;`

Exemplos:
- Variável: `myVariable`
- Função: `calculateTotal()`
- Comando: `npm install`

---

## Exercício Prático

Tente criar um parágrafo usando:
1. **Negrito** para destacar termos importantes
2. *Itálico* para ênfase
3. `Código inline` para comandos ou variáveis
4. [Links] para referências externas

O **PDI Learning Hub** é um *sistema moderno* de aprendizado que utiliza `Markdown` para criar conteúdo rico. Visite nossa [documentação](https://github.com) para mais detalhes.
